# Игра "Угадай число"

Проект представляет собой простую игру, в которой компьютер сам загадывает число от 1 до 100 и сам пытается его угадать.

## Что делает программа

Программа проверяет, за сколько попыток алгоритм угадывает случайное число.

Для проверки используется 1000 случайных чисел. После этого считается среднее количество попыток.

## Алгоритмы

В проекте есть функция случайного угадывания:

- компьютер выбирает случайное число от 1 до 100;
- сравнивает его с загаданным числом;
- повторяет попытки, пока не угадает.

Также можно улучшить программу с помощью бинарного поиска. Такой алгоритм угадывает число намного быстрее, потому что каждый раз сужает диапазон поиска.
