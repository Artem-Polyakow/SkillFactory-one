# NumPy: бизнес-задачи с исходными данными

В этом наборе лежат CSV-файлы для самостоятельной тренировки NumPy.
Решения в файлы не включены.

## Как загрузить CSV в NumPy

Вариант 1 - через pandas, затем взять NumPy-массив:

```python
import pandas as pd
import numpy as np

df = pd.read_csv("task_02_sales_plan_fact.csv")
arr = df[["sales_plan_rub", "sales_fact_rub"]].to_numpy()
```

Вариант 2 - напрямую через NumPy для полностью числовых колонок:

```python
import numpy as np

arr = np.genfromtxt(
    "task_10_missing_payments.csv",
    delimiter=",",
    names=True,
    dtype=None,
    encoding="utf-8"
)
```
